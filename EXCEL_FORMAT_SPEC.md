# 📋 Excel 報價單格式說明

## 🎯 **根據您的說明，正確的格式是：**

### **固定位置**
- **B9** = 客戶名稱
- **F9** (大約) = 報價單號碼
- **B13** = 設備名稱

### **明細資料（從B14開始）**
- **B 欄** = 待校正項目（熱電偶溫度線、數位電力計(單相)、比流器等）
- **D 欄** = 數量
- **E 欄** = 單價

---

## 📊 **範例**

| 行 | A | B | C | D | E | F |
|----|---|---|---|---|---|---|
| **9** | | **財團法人工業技術研究院** | | | | **F-26200** |
| **13** | | **電冰箱及多功能性能試驗室** | | | | |
| **14** | | 熱電偶溫度線 | | 2 | 6000 | |
| **15** | | 數位電力計(單相) | | 1 | 8000 | |
| **16** | | 比流器 | | 4 | 4000 | |

---

## ✅ **提取結果**

```
報價單號: F-26200 (從 F9)
客戶: 財團法人工業技術研究院 (從 B9)
設備名稱: 電冰箱及多功能性能試驗室 (從 B13)

明細:
1. 熱電偶溫度線 | 數量:2 | 單價:$6,000
2. 數位電力計(單相) | 數量:1 | 單價:$8,000
3. 比流器 | 數量:4 | 單價:$4,000
```
