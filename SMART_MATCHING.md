# 🔍 智能產品匹配功能

## ✨ 新功能說明

系統現在會**智能匹配現有產品**，避免重複建立相似的產品！

---

## 🎯 工作原理

### 匹配流程：

```
1. 精確匹配
   ↓ (找不到完全相同)
2. 模糊匹配 - 搜尋所有產品
   ↓
3. 計算相似度 (0-100%)
   ↓
4. 根據相似度決定動作
```

---

## 📊 相似度規則

| 相似度 | 動作 | 說明 |
|--------|------|------|
| **≥ 85%** | ✅ 自動使用現有產品 | 幾乎相同，自動匹配 |
| **70-84%** | ❓ 詢問用戶 | 顯示建議，由您決定 |
| **50-69%** | ➕ 建立新產品 | 相似度不夠高 |
| **< 50%** | ➕ 建立新產品 | 差異太大 |

---

## 💡 實際範例

### 範例 1: 高相似度（自動匹配）

**Excel 中的產品**: `交流電壓表`  
**資料庫中已有**: `交流電壓錶`  
**相似度**: 92%  
**動作**: ✅ 自動使用現有產品

```
      🔍 找到相似產品:
         新產品: 交流電壓表
         相似度: 92%
         現有: 交流電壓錶
         規格: 0-300V
         價格: $3,000
         ✓ 自動使用現有產品 (相似度≥85%)
```

### 範例 2: 中等相似度（詢問用戶）

**Excel 中的產品**: `白金電阻溫度計（三用電錶）`  
**資料庫中已有**: `白金電阻溫度計`  
**相似度**: 75%  
**動作**: ❓ 詢問用戶

```
      🔍 找到相似產品:
         新產品: 白金電阻溫度計（三用電錶）
         相似度: 75%
         現有: 白金電阻溫度計
         規格: 校正點0-100℃
         價格: $5,000
         使用現有產品? (y/N): _
```

您可以選擇：
- 輸入 `y` → 使用現有產品
- 輸入 `N` → 建立新產品

### 範例 3: 低相似度（建立新產品）

**Excel 中的產品**: `工時2天`  
**資料庫中已有**: `工時1天`  
**相似度**: 62%  
**動作**: ➕ 直接建立新產品

```
      ➕ 新增產品: 工時2天
```

## ✅ 優勢

### 1️⃣ **避免重複**
不會建立 `電壓表`、`電壓錶`、`電壓表計` 等相似產品

### 2️⃣ **統一命名**
自動使用資料庫中已有的標準名稱

### 3️⃣ **保持價格一致**
相似產品會使用現有的標準價格

### 4️⃣ **智能建議**
顯示相似產品的完整資訊，幫助您決策

## 📋 使用範例

### 情境：匯入新報價單

您的 Excel 中有：
- `白金測溫計`
- `三用電表`
- `工時費用`

資料庫中已有：
- `白金電阻溫度計`（規格：0-100℃，$5,000）
- `三用電錶`（規格：Fluke 179，$2,500）

執行匯入時：

```
      🔍 找到相似產品:
         新產品: 白金測溫計
         相似度: 72%
         現有: 白金電阻溫度計
         規格: 0-100℃
         價格: $5,000
         使用現有產品? (y/N): y
         ✓ 使用現有產品

      🔍 找到相似產品:
         新產品: 三用電表
         相似度: 88%
         現有: 三用電錶
         規格: Fluke 179
         價格: $2,500
         ✓ 自動使用現有產品 (相似度≥85%)

      ➕ 新增產品: 工時費用
```

結果：
- ✅ 使用現有的「白金電阻溫度計」
- ✅ 自動使用「三用電錶」
- ➕ 新增「工時費用」

## 💡 最佳實踐

### 1. **統一產品名稱**
在 cali_products 中維護標準的產品名稱

### 2. **定期檢查**
匯入後檢查產品列表，合併相似項

### 3. **使用建議**
相似度 70% 以上的建議通常是準確的

### 4. **手動調整**
如果匹配不準確，事後可在系統中手動調整
