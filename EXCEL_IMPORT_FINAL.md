# ✅ Excel 報價單自動匯入 - 最終版

## 🎯 **優化完成！**

### ✨ **新功能特色**

#### 1️⃣ **精準欄位提取**
- ✅ **B9** → 客戶名稱
- ✅ **B 欄** → 產品名稱
- ✅ **D 欄** → 數量
- ✅ **E 欄** → 單價
- ✅ **合計** → 自動計算（數量 × 單價）

#### 2️⃣ **智能過濾**
- ✅ **只提取有數量和價格的列**
- ✅ 自動跳過標題和說明文字
- ✅ 過濾無效資料

#### 3️⃣ **智能產品匹配**
- ✅ **≥85% 相似度** → 自動使用現有產品
- ✅ **70-84% 相似度** → 詢問是否使用
- ✅ **<70% 相似度** → 建立新產品

---

## 🚀 **使用方式**

### **只需 2 步驟**：

```powershell
# 步驟 1: 放入 Excel
# 將報價單放入 excel_quotes/ 資料夾

# 步驟 2: 執行匯入
python smart_excel_import.py
```

---

## 📊 **處理流程**

```
1. 掃描 Excel 檔案
   ↓
2. 從 B9 提取客戶名稱
   ↓
3. 從 B/D/E 欄提取產品、數量、單價
   ↓
4. 過濾：只保留有數量和價格的列
   ↓
5. 智能匹配現有產品（避免重複）
   ↓
6. 匯入到 Supabase
   ↓
7. 完成！✅
```

## 💡 **實際範例**

### 您的 Excel 格式：

| 欄位 | A | B | C | D | E | F |
|------|---|---|---|---|---|---|
| **9** | - | **財團法人工業技術研究院** | - | - | - | - |
| **10** | 項次 | 品名 | 規格說明 | 數量 | 單價 | 合計 |
| **11** | 1 | 白金電阻溫度計 | 0-100℃ | 28 | 5000 | 140000 |
| **12** | 2 | 三用電錶 | Fluke 179 | 4 | 6500 | 26000 |
| **13** | - | 備註說明文字 | - | - | - | - |

### 系統提取結果：

```
✓ 從 B9 提取客戶: 財團法人工業技術研究院
✓ 提取 2 筆明細

明細預覽:
  - 白金電阻溫度計 | 數量:28 | 單價:$5,000
  - 三用電錶 | 數量:4 | 單價:$6,500
```

**自動跳過**：
- ❌ 第 13 列（無數量和價格）
- ❌ 標題列
- ❌ 備註文字
