# ✅ Excel 報價單自動匯入 - 最終完成版

## 🎉 **已完成！根據您的報價單格式優化**

---

## 📊 **欄位對應**

### **固定位置**
| Excel 位置 | 內容 | Supabase 欄位 |
|-----------|------|--------------|
| **B9** | 客戶名稱 | `customer_name` |
| **F9** | 報價單號 | `order_number` |
| **B13** | 設備名稱 | `equipment_name` |

### **明細資料（從第14列開始）**
| Excel 欄位 | 內容 | Supabase 欄位 |
|-----------|------|--------------|
| **B 欄** | 校正項目 | `product_name` |
| **C 欄** | 設備案號（可選） | `equipment_number` |
| **D 欄** | 數量 | `quantity` |
| **E 欄** | 單價 | `unit_price` |

---

## 📋 **範例**

### 您的 Excel 格式：
```
B9:  財團法人工業技術研究院
F9:  F-26200
B13: 電冰箱及多功能性能試驗室

第14列: 熱電偶溫度線    | 數量:2  | 單價:6000
第15列: 數位電力計(單相) | 數量:1  | 單價:8000
第16列: 比流器          | 數量:4  | 單價:4000
```

### 系統提取結果：
```
✓ B9 客戶: 財團法人工業技術研究院
✓ F9 報價單號: F-26200  
✓ B13 設備: 電冰箱及多功能性能試驗室

📋 掃描明細（第14列起）...
✓ 提取 3 筆明細

📋 明細預覽:
  1. 熱電偶溫度線 | 2個 | $6,000
  2. 數位電力計(單相) | 1個 | $8,000
  3. 比流器 | 4個 | $4,000
```

## 🔍 **提取邏輯說明**

### **必須條件**：
```
✅ B欄有校正項目名稱
✅ D欄有數量（數字）
✅ E欄有單價（數字 > 0）
```

### **自動跳過**：
```
❌ 空白列
❌ 標題列（項目、品名、數量、單價等關鍵字）
❌ 說明文字
❌ 沒有數量或價格的列
```
